Comandos del Angular cli
    Con este comando instalamos angular utilizando npm
  1. npm install -g @angular/cli
    2. Comando pa crear un proyecto en angular
  2. ng new nombdelProyect
    3. Para generar un puerto para verlo en le navegador hacemos esto
  3. ng serve si no arranca hacemos un npm install y volvemos a intentar

    4. Comando para crear modulos
  4. ng g m nombredelmodulo

    5. Comando pa crear componentes c significa componente, y s servicios

    Comando para crear un Componente
  5. ng g c pages/paises
    6. Comando para crear Servicios
  6. ng g s pages/paises
    7. tambien podemos decirle que nos cree una carpeta service, para no crearla manualmente eje:
  7. ng g s gifs/services/gifs
    8. Comando para las rutas
  8. ng g m AppRouting --flat
    9. De esta manera integramos pwa a angular
  9. ng add @angular/pwa
    10. Para llevar la aplicacion a producion hacemos esto
  10. ng build --configuration production
